## Required UI attribution

앱 하단 또는 정보 패널에 아래 항목을 표시합니다.

- Cross-reference data source
- Bible text translation/source
- Project repository link
- License notes

라이선스가 불명확한 데이터는 공개 배포에 포함하지 않습니다.
